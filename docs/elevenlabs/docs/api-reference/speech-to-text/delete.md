# Source: https://elevenlabs.io/docs/api-reference/speech-to-text/delete

Delete a previously generated transcript by its ID.

### Path parameters

transcription\_idstringRequired

The unique ID of the transcript to delete

### Headers

xi-api-keystringOptional

### Response

Delete completed successfully.

### Errors

401

Transcripts Delete Request Unauthorized Error

422

Transcripts Delete Request Unprocessable Entity Error
